# MGC (A Modpack for Getting Over It, developed with C#) 🧗
## 📜 项目简介
**MGC** 是基于《Getting Over It with Bennett Foddy》开发的非商业模组包，使用 C# 和 Unity 开发，通过反编译工具 dnSpy 实现游戏机制扩展。
